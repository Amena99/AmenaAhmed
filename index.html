'Hungry' Restaurant Locator

Makes creative use of the Zomato API and JavaScript Geolocation.

Going out for a bite with friends? Need a restaurant catered to your group's specific needs? 'Hungry' can help! This app tracks the user's location (latitude & longitude) using JavaScript's geolocation and then uses it to find restaurants within a 2/3 mile radius that match the user's preference. In the mood for Thai? Need vegan? Gluten free? Just delegate the search to this helpful app!

  • Built: December 2018

'Devvo' Learning Platform

Uses MySQL to provide the user with structured tasks to prepare for developer interviews.

There are a lot of web developers like me out there prepping for interviews or just intending to learn. Learning on your own can be difficult and finding the right balance of activities can be a challenge! 'Devvo' makes this job easier by providing the user with an organized platform that breaks down the big learning venture into manageable tasks that can be completed one at a time. Lesson plans are stored in a MySQL database which is coded using Sequelize ORM. A dynamically populating progress bar and successboard charts using Chart.js keep the motivations high!

  • Built: February 2019
  • Click here for the live website.
  • Click here to see my code on GitHub.
